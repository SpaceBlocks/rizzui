# Flex

Flex is a layout component that allows you to create a FlexBox layout.

```tsx
import { Flex } from "rizzui";
```

<div className="mb-20" />

## Default

The style of **Flex** component. The **Flex** gets the default style of the css flex property.

<Paper>
  <FlexLayout
    justify="center"
    className="[&>div]:size-24 [&>div]:text-3xl"
  />
</Paper>

```tsx
import { Flex, Box } from "rizzui";

export default function App() {
  return (
    <Flex>
      {[1, 2, 3].map((i) => (
        <Box
          key={i}
          className="border border-muted rounded-md size-24 text-3xl font-semibold flex justify-center items-center shadow"
        >
          {i}
        </Box>
      ))}
    </Flex>
  );
}
```

## With Justify Content

The style of **Flex** component can be modified using the **justify** prop. The **justify** prop is used to changed the position of flex items inside the flex box.

<Paper>
  <FlexLayout className="mb-10" />
  <FlexLayout
    justify="center"
    className="mb-10"
  />
  <FlexLayout justify="end" />
</Paper>

```tsx
import { Flex, Box } from "rizzui";

export default function App() {
  return (
    <>
      <Flex>
        {[1, 2, 3].map((i) => (
          <Box
            key={i}
            className="border border-muted rounded-md size-10 text-3xl font-semibold flex justify-center items-center shadow"
          >
            {i}
          </Box>
        ))}
      </Flex>

      <Flex justify="center">
        {[1, 2, 3].map((i) => (
          <Box
            key={i}
            className="border border-muted rounded-md size-24 text-3xl font-semibold flex justify-center items-center shadow"
          >
            {i}
          </Box>
        ))}
      </Flex>

      <Flex justify="end">
        {[1, 2, 3].map((i) => (
          <Box
            key={i}
            className="border border-muted rounded-md size-24 text-3xl font-semibold flex justify-center items-center shadow"
          >
            {i}
          </Box>
        ))}
      </Flex>
    <>
  );
}
```

## With Align Items

The style of **Flex** component can be modified by using the **align** prop. The **align** prop is used to align the flex items inside the flex box.

<Paper>
  <FlexLayoutAlign className="mb-10" />
  <FlexLayoutAlign
    align="center"
    justify="center"
    className="mb-10"
  />
  <FlexLayoutAlign
    align="end"
    justify="end"
  />
</Paper>

```tsx
import { Flex, Box } from "rizzui";

export default function App() {
  return (
    <>
      <Flex>
        {[1, 2, 3].map((i) => (
          <Box
            key={i}
            className="border border-muted rounded-md w-16 text-xl font-semibold flex justify-center items-center shadow"
            style={{ height: `${(i + 1) * 20}px` }}
          >
            {i}
          </Box>
        ))}
      </Flex>

      <Flex justify='center' align='center'>
        {[1, 2, 3].map((i) => (
          <Box
            key={i}
            className="border border-muted rounded-md w-16 text-xl font-semibold flex justify-center items-center shadow"
            style={{ height: `${(i + 1) * 20}px` }}
          >
            {i}
          </Box>
        ))}
      </Flex>

      <Flex justify='end' align='end'>
        {[1, 2, 3].map((i) => (
          <Box
            key={i}
            className="border border-muted rounded-md w-16 text-xl font-semibold flex justify-center items-center shadow"
            style={{ height: `${(i + 1) * 20}px` }}
          >
            {i}
          </Box>
        ))}
      </Flex>
    <>
  );
}
```

### Flex Props

Here is the API documentation of the **Flex** component. And the rest of the props are the same as the original html element. You can use props like **id**, **title**, **onClick** etc.

<div className="overflow-x-auto">

|   Props   |                 Type                  |                    Description                     |  Default  |
| :-------: | :-----------------------------------: | :------------------------------------------------: | :-------: |
|    as     |       `div` or any html element       |                     Render as                      |  `"div"`  |
| children  |           `React.ReactNode`           |        Accepts everything React can render         |   \_\_    |
| direction | [**Direction Type**](#direction-type) |      Defines the flex direction in a flex box      |  `"row"`  |
|  justify  |   [**Justify Type**](#justify-type)   | Defines the position of flex items inside flex box | `"start"` |
|   align   |     [**Align Type**](#align-type)     |               Aligns the flex items                | `"start"` |
|    gap    |       [**Gap Type**](#gap-type)       |     Defines the gap between columns in a grid      |   `"4"`   |
| className |               `string`                |        Add custom classes to the component         |   \_\_    |

</div>
