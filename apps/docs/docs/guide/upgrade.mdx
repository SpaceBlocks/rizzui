# Upgrade to v0.8.5

This guide will help you upgrade your existing rizzui project to v0.8.5.

## Upgrade Steps

### Step 1: Update the `RizzUI` package

```bash
npm install rizzui@latest
```

### Step 2: Update the `HeadlessUI` package

```bash
npm install @headlessui/react@latest
```

### Step 3: Update the `Floating UI` package

```bash
npm install @floating-ui/react@latest
```
