# Upgrade

## Upgrade to v0.8.5

This guide will help you upgrade your existing `rizzui` project to v0.8.5.

:::info
We have handled all the breaking changes inside `rizzui` and its dependencies. So, you don't need to worry about breaking changes.
:::

## Upgrade Steps

##### Update the `RizzUI`, `HeadlessUI`, `Floating UI` packages to the latest version

```bash
npm install rizzui@latest @headlessui/react @floating-ui/react
```

_* **Note:** You have the flexibility to use alternative package managers such as **bun**, **pnpm**, and **yarn**.*_
